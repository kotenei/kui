# API

```jsx
import { Timeline, TimelineItem } from 'kui-react';
```

## Timeline

| 名称      | 类型     | 默认值       | 描述       |
| --------- | -------- | ------------ | ---------- |
| prefixCls | `string` | 'k-timeline' | 组件样式名 |

## TimelineItem

| 名称  | 类型                          | 默认值    | 描述                                                           |
| ----- | ----------------------------- | --------- | -------------------------------------------------------------- |
| color | `string`                      | 'primary' | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
| dot   | `React.ReactNode` \| `string` | -         | 圆点内容                                                       |
