# API

```jsx
import { Avatar } from 'kui-react';
```

## Avatar

| 名称      | 类型            | 默认值   | 描述                                                     |
| --------- | --------------- | -------- | -------------------------------------------------------- |
| prefixCls | string          | k-avatar | 组件样式名                                               |
| icon      | React.ReactNode | -        | 图标                                                     |
| square    | boolean         | false    | 是否正方形                                               |
| size      | string          | md       | 可选值 'sm' 'lg' 'md'                                    |
| color     | string          | -        | 颜色，可选 'primary' 'info' 'success' 'warning' 'danger' |
| src       | string          | -        | 图标路径                                                 |
