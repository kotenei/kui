# API

```jsx
import { Mask } from 'kui-react';
```

## Mask

| 名称           | 类型     | 默认值 | 描述               |
| -------------- | -------- | ------ | ------------------ |
| zIndex         | number   | 800    | 层叠               |
| show           | boolean  | false  | 是否显示           |
| timeout        | number   | 300    | 动画超时时间(毫秒) |
| transitionName | string   | 'fade' | 动画名称           |
| onClick        | ()=>void | -      | 点击回调           |
